## Introduction

This project implements the BGLOW website. It addresses the need for a functional and responsive online presence for the BGLOW brand.

The primary benefits include a modern user interface built with CSS, a responsive design for optimal viewing across devices, and a readily deployable website. This allows for easy content updates and future feature additions.

## Prerequisites

To successfully build and deploy the BGLOW website, ensure the following prerequisites are met.

**Required Software:**

*   **Web Browser:** A modern web browser such as Chrome, Firefox, or Safari.
*   **Text Editor:** A text editor or Integrated Development Environment (IDE) capable of handling CSS files. Examples include Visual Studio Code, Sublime Text, or Atom.

**Optional Software:**

*   **Version Control System:** Git (version 2.28.0 or later) is recommended for managing code changes.
*   **Code Editor Extensions:** Consider installing extensions for your code editor to enhance CSS development, such as linters and formatters.

## Installation

To install and set up the BGLOW website, follow these steps:

1.  **Clone the Repository:** Use `git clone` to obtain the project files.

    ```bash
    git clone https://github.com/jephter-olamiposi/BGLOW.git
    ```

2.  **Navigate to the Project Directory:** Change your current directory to the project's root.

    ```bash
    cd BGLOW
    ```

3.  **Install Dependencies:** Install the necessary Node.js packages using npm.

    ```bash
    npm install
    ```

4.  **Environment Variable Setup:** Create a `.env` file in the project's root directory. Define the required environment variables.  For example:

    ```
    PORT=3000
    NODE_ENV=development
    ```

## Usage

To run the BGLOW website, navigate to the project directory in your terminal and execute the following command:

```bash
npm start
```

This command starts the development server, typically making the website accessible at `http://localhost:3000`.
